üß™ Testing Checklist

HTML & Structure

  • Page displays without errors
  • All sections are visible (Home, About, Skills, Projects, Contact)
  • Navigation menu works and links to sections
  • Semantic HTML is properly structured
  • Meta tags are present

Styling & Responsiveness

  • Website looks good on desktop (1200px+)
  • Website looks good on tablet (768px)
  • Website looks good on mobile (480px)
  • Colors are properly applied
  • Animations and transitions are smooth
  • Fonts are readable and properly sized

Form Validation

  • Try submitting empty form - shows required field errors
  • Try invalid email - shows email validation error
  • Try short message - shows length error
  • Fill form correctly - form submits successfully
  • Check success message appears
  • Form data saves to localStorage

PHP & Database

  • Contact form submits successfully
  • Data appears in admin_contacts.php
  • Database tables are created
  • phpMyAdmin shows contact messages
  • No PHP errors in browser console

JavaScript Features

  • Navigation links scroll smoothly
  • Form validation happens in real-time
  • Error messages appear when needed
  • Intersection Observer animations work
  • Console has no JavaScript errors

API Testing

  • GET /php/api_projects.php?action=featured works
  • GET /php/api_projects.php?action=all works
  • JSON responses are properly formatted
  • Error handling returns proper status codes
